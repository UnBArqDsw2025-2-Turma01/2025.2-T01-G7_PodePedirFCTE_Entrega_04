# 4.1.2. Visualização do Processo

## Introdução

A visualização de processo lida com os aspectos dinâmicos do sistema, descreve processos de sistema e como eles se comunicam e enfatiza o comportamento do sistema durante sua execução, descrevendo diferentes atividades de sistema. Diagramas de sequência e de atividades são utilizados para a visualização de processo.

## Visão Geral

O PodePedirFCTE é um sistema que permite que os usuários, estudantes da UnB campus Gama, possam fazer pedidos de delivery para estabelecimentos próximos à faculdade. Dessa forma, os alunos possuem um novo conjunto de opções para seus almoços e lanches além de fomentar o comércio local.
